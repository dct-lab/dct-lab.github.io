# Openings

**We are currently open for PhD and Master applications. If you are interested in pursuing a Master/PhD degree at IIE/UCAS, see [master/phd application in iie](http://www.iie.cas.cn/xsjy2020/zsxx2020/sszs2020/).** 

### 涂碧波

**Research Field**

Arithmetic infrastructure security: Focus on the cutting-edge technology of arithmetic infrastructure (hyper-converged virtualization, cloud-network convergence, cloud-network edge-end collaboration, etc.) and security requirements, and carry out systematic endogenous security research at the network, system and data levels.

- Trusted computing environment: Research on the security and trustworthiness mechanism and application transparent and confidential computing architecture of object-end operating system, virtualized operating system and cloud platform operating system based on secure and trustworthy chips, and provide key technologies and systems of trusted computing with software and hardware collaboration.
- Software-defined boundary: Research on dynamic trust decision-making mechanisms such as continuous identity authentication, trusted device management, network traffic interpretable intelligent analysis and intention-driven software-defined network security architecture, providing dynamic trust enabling technologies and practicing zero trust.
- Flexible data protection: Research on cloud-network-end collaborative data environment cryptographic service architecture and its enabling differentiated data security services to provide flexible data protection technology and data governance support platform for diverse data security scenarios.

**Admissions Information**

- Direction 1: Computer Science and Technology - Computer Software and Theory - Operating Systems

- Direction 2: Cyberspace Security - Network and System Security - Information Protection Technology

**Enrollment requirements: motivated, willing to learn, hands-on, good at teamwork, and enjoy academic communication.** 

**Courses Taught**

- Operating System Security 40 hours UCAS Spring Semester
